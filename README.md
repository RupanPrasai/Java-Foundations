# Java Foundations

This repository contains my work and notes for learning Java through the University of Helsinki Java MOOC (Java Programming I & II).

## Structure

```
java/
├── mooc-java-1/
├── mooc-java-2/
├── notes/
└── scratch/
```

- **mooc-java-1/** → Java Programming I exercises  
- **mooc-java-2/** → Java Programming II exercises  
- **notes/** → Concepts, patterns, and summaries  
- **scratch/** → Temporary experiments / test code  

---

## Goals

- Build a strong foundation in Java

- Develop problem-solving and algorithmic thinking

## Tech Stack

- Java (JDK 17+)

## Setup

### 1. Install Java
Ensure you have a compatible JDK installed:
```bash
java -version
```

### 2. Clone repo
```bash
git clone <repo-url>
cd java
```

### 3. Navigate to a course section
```bash
cd mooc-java-1
```
